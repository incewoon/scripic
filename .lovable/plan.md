## 변경 내용

### `src/routes/create.tsx` — 프리셋 줄에 커스텀 태그 이어붙이기

- 마운트 시 `getAlbums()`로 저장된 앨범을 한 번 로드 → 모든 앨범의 `tags`를 평탄화해서 **프리셋(여행/가족/일상/친구/음식/특별한날)에 속하지 않은 것**만 모아 `myTags` 상태에 저장.
  - 정렬 기준: 빈도 X, **처음 등장한 순서**로 누적(중복 제거, 대소문자 무시).
- 기존 가로 스크롤 칩 줄(`flex gap-1.5 overflow-x-auto`)을 그대로 두고, **프리셋 칩들 바로 오른쪽에** `myTags` 칩을 같은 컴포넌트 스타일로 이어서 렌더링.
  - 동작은 프리셋과 동일: 탭하면 `tags`에 토글, 5개 제한 유지, 선택 시 그라데이션/미선택 시 외곽선.
  - 줄바꿈 없음: 부모의 `whitespace-nowrap flex-shrink-0`가 그대로 적용되어 우측 스크롤만 발생.
- 현재 입력 세션에서 새로 만든 커스텀 태그(`customTags`)는 기존 로직 그대로 — 이미 활성 상태로 보이고, 저장 후 다음 생성 화면에서 `myTags`로 합류.

### 비변경

- 두 번째 줄(현재 세션의 커스텀 활성 태그 표시), 입력창/추가 버튼, i18n, 메인 화면, 저장 형식 모두 그대로.
- 프리셋 목록·5개 제한·디자인 토큰 변경 없음.
