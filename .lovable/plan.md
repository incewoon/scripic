## Goal
설정 페이지의 "앨범 백업" 섹션에 사용자가 안심하고 백업을 사용할 수 있도록 안내 문구를 보강한다.

## 변경 파일
1. **`src/lib/i18n.ts`** — 영문/한글 안내 문자열 추가
2. **`src/routes/settings.tsx`** — 백업 섹션 내부에 안내 카드 렌더링 (다운로드/복원 버튼 위)

## 추가할 안내 내용 (한글 기준, 영문도 동일 톤으로)

**📱 기기 저장만 지원**
백업 파일은 사용자의 기기(다운로드 폴더)에만 저장됩니다. 보안과 개인정보 보호를 위해 클라우드 백업은 지원하지 않습니다. 사진과 앨범은 모두 사용자 기기 안에서만 관리돼요.

**🔐 PIN 암호화**
모든 백업 파일은 사용자가 설정한 PIN으로 암호화되어 저장됩니다. PIN을 잃어버리면 복원할 수 없으니 꼭 기억해 주세요.

**🧭 백업 과정**
1. "백업 파일 다운로드" 버튼을 누르면 PIN 입력창이 나타납니다.
2. PIN을 입력하면 모든 앨범·사진·테마 설정이 암호화된 `.moarabak` 파일로 만들어져 기기에 저장됩니다.
3. 복원할 때는 "백업 파일에서 복원"을 누르고 같은 PIN으로 파일을 선택하면 됩니다.

**⚠️ 주의**
앱 데이터를 삭제하거나 브라우저 저장소를 비우면 백업 파일 없이는 복원할 수 없으니, 정기적으로 백업 파일을 안전한 곳에 보관해 주세요.
