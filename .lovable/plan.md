## create.tsx 사진 추가 UI 정리

### 1. 진행률 바 제거
사진 그리드 상단의 `h-1.5 bg-muted/70 rounded-full` 진행률 바 및 관련 `pct` 계산식 삭제.

### 2. 하단 "사진 추가" 버튼 제거
하단 CTA 영역에서 `items.length > 0 && items.length < PHOTO_MAX` 조건의 "사진 추가" 버튼 블록 제거. 하단에는 항상 "AI와 이야기하기" 버튼만 남김.

### 3. 그리드 빈 슬롯에 정사각형 "사진 추가" 버튼 표시
사진 영역(슬롯 1, 2, 3)에 남은 빈 칸 중 **가장 앞(가장 작은 번호) 한 칸**에만 점선 테두리 정사각형 "사진 추가" 버튼 표시, 나머지 빈 칸은 비워둠.

- items 0장 → 슬롯 1에만 점선 버튼+"사진 추가" 라벨, 슬롯 2·3 비움 (기존 동작과 동일)
- items 1장 → 슬롯 2에만 점선 버튼+"사진 추가" 라벨, 슬롯 3 비움
- items 2장 → 슬롯 3에만 점선 버튼+"사진 추가" 라벨
- items 3장 → 모든 슬롯이 사진으로 채워짐, 추가 버튼 없음

버튼 스타일은 기존과 동일: `aspect-square rounded-2xl border-2 border-dashed border-primary/40`, `tryOpenPicker` 호출. SortableContext에 포함되지 않도록 일반 `<button>`로 그리드에 삽입.

### 4. 기타 변경 없음
i18n, 백업, 다른 화면/기능 수정 없음.
